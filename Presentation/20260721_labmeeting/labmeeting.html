What’s GitHub?

GitHub is a web-based code cloud

: Once you have done your job, you can upload your code/files to GitHub, so others can access it.


Storage limit

  • 100 MB per file
  • ~ 2GB per repository (ideally)
  • Unlimited repo (No strict rules…)
